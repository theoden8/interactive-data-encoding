## Tools

### Compression

* C++14
* Qt

### Error correction

* C++14
* nuklear, glfw3, glew
* gmpxx

## Usage

### Prerequisites

Make sure you have everything installed.

Then download [nuklear](https://github.com/vurtun/nuklear) into **correction/** folder.

### Compiling

	make

### Testing

	make test
	
### Cleaning
		
	make clean
